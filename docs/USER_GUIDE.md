## üõí Buyer Guide

### Browsing Products

**Home Page:**
- View featured products

**Product Catalog:**
1. Click "Products" in navigation
2. Use filters to narrow results:

   - Newest first

**Product Details:**
- View high-quality product images
- Read detailed descriptions
- Check nutritional information
- See farmer information and ratings
- Read customer reviews

### Shopping Cart

**Adding Items:**
1. On product page, select quantity
2. Click "Add to Cart"
3. Continue shopping or proceed to checkout

**Managing Cart:**
1. Click cart icon in navigation
2. Adjust quantities
3. Remove unwanted items
4. View total cost including delivery

### Placing Orders

**Checkout Process:**
1. Review cart items
2. Confirm delivery address
3. Select delivery date and time
4. Add special instructions (optional)
5. Review order summary
6. Click "Place Order"

**Order Confirmation:**
- You'll receive order confirmation email
- Order appears in your "My Orders" section
- Track order status in real-time

### Order Management

**Order Statuses:**
- **Pending**: Waiting for farmer confirmation
- **Accepted**: Farmer confirmed, preparing order
- **Ready**: Order ready for pickup/delivery
- **Completed**: Order delivered successfully
- **Cancelled**: Order cancelled by farmer or buyer

**Tracking Orders:**
1. Go to "My Orders"
2. Click on specific order
3. View detailed status and timeline
4. Contact farmer if needed

### Messaging Farmers

**Starting Conversation:**
1. Go to product page or farmer profile
2. Click "Contact Farmer"
3. Write your message
4. Click "Send Message"

**Managing Messages:**
1. Go to "Messages" section
2. View all conversations
3. Reply to farmer messages
4. Mark messages as read

### Writing Reviews

**After Order Completion:**
1. Go to "My Orders"
2. Find completed order
3. Click "Write Review"
4. Rate the farmer (1-5 stars)
5. Rate specific aspects:
   - Product quality
   - Delivery service
   - Communication
6. Write detailed comment
7. Submit review

**Review Guidelines:**
- Be honest and constructive
- Focus on product quality and service
- Avoid personal attacks
- Reviews can be edited before approval

**Managing Existing Products:**
1. View all products in "My Products"
2. Edit product details
3. Update stock quantities
4. Adjust prices

6. Delete products if needed

**Product Photography Tips:**
- Use natural lighting
- Show product from multiple angles
- Include size reference (coins, hands)
- Ensure images are clear and high quality
- Show any certifications or labels

## üîß Troubleshooting

### Common Issues

**Login Problems:**
- **Issue**: Can't log in
- **Solutions**:
  - Check email and password spelling
  - Ensure caps lock is off
  - Try password reset
  - Clear browser cache
  - Check internet connection

**Page Loading Issues:**
- **Issue**: Pages load slowly or not at all
- **Solutions**:
  - Check internet connection
  - Refresh the page
  - Clear browser cache and cookies
  - Try different browser
  - Disable browser extensions

**Order Issues:**
- **Issue**: Order not appearing
- **Solutions**:
  - Check "My Orders" section
  - Verify order confirmation email
  - Contact farmer directly
  - Check spam folder for emails
  - Contact support if needed

**Payment Problems:**
- **Issue**: Payment not processing
- **Solutions**:
  - Verify payment information
  - Check bank account balance
  - Try different payment method
  - Contact your bank
  - Contact platform support

### Browser-Specific Issues

**Chrome:**
- Clear browsing data: Settings ‚Üí Privacy ‚Üí Clear browsing data
- Disable extensions: Settings ‚Üí Extensions
- Reset Chrome: Settings ‚Üí Advanced ‚Üí Reset

**Firefox:**
- Clear cache: Options ‚Üí Privacy ‚Üí Clear Data
- Disable add-ons: Add-ons ‚Üí Extensions
- Refresh Firefox: Help ‚Üí Troubleshooting Information

**Safari:**
- Clear cache: Safari ‚Üí Preferences ‚Üí Privacy ‚Üí Manage Website Data
- Disable extensions: Safari ‚Üí Preferences ‚Üí Extensions

### Mobile Issues

**App Installation:**
- **Issue**: Can't install PWA
- **Solutions**:
  - Use supported browser (Chrome, Safari)
  - Ensure sufficient storage space
  - Check browser settings
  - Try manual bookmark creation

**Touch Issues:**
- **Issue**: Buttons not responding
- **Solutions**:
  - Clean screen
  - Remove screen protector temporarily
  - Try different finger/stylus
  - Restart browser
